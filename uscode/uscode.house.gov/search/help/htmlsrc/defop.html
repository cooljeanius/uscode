Understanding the Default Operator

Unless you enter an explicit logical search operator (i.e., OR, AND, NOT, W/n, ADJ, NEAR) in your query, PLWeb Turbo will interpret the query using a predetermined default operator.

The default operator is OR; it is automatically used in the absence of any manually entered operators. If you enter a multi-word query, PLWeb Turbo will retrieve all records that contain at least one of the words. For example, the following query: 

tribal politics in South Africa
would be interpreted as: 

tribal OR politics OR South OR Africa
Any record in the current database that contains one or more of those terms would satisfy the conditions of the search and therefore, be retrieved.
